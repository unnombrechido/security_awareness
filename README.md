# Sitio de Entrenamiento en Seguridad de la Información

Este repositorio contiene un sitio web simple para un curso de seguridad de la información de 15 minutos y una evaluación de 2-5 minutos, diseñado para empleados de PYMEs en México.

## Instrucciones para Empleados
- Accede al curso en la página principal (index.html).
- Navega las diapositivas con flechas o clics.
- Al final, haz clic en el enlace para la evaluación (quiz.html).
- El quiz es autocalificado; no envía datos.

## Configuración Técnica (para admins)
- Hospedado en GitHub Pages.
- Usa Reveal.js para diapositivas (CDN externo, sin dependencias locales).
- Si actualizas, commitea cambios y GitHub Pages se refresca automáticamente.

Contacto: Si tienes dudas, contacta al departamento de TI o al proveedor de seguridad.
